CascadeV4 is a high-level dynamic programming language, revamped for the 4th time to make more extensive use of OOP. To run any file, execute main.py.

Lexer

Inputs: Program File

Outputs: Token Stream

Parser

Inputs: Token Stream

Outputs: AST

Interpreter

Inputs: AST

Outputs: Execution of AST
